# 择时因子指标信号评分体系

## 一、评分体系概述

本评分体系用于对择时因子的指标信号进行综合评价，采用**多维度加权评分法**，各维度评分范围为 **[0, 1]**，最终合成综合评分并划分等级。

### 1.1 权重配置

| 维度 | 权重 | 说明 |
|------|------|------|
| IC评分 | 35% | 因子预测能力的相关性检验 |
| 信号评分 | 35% | 交易信号的实际效果检验 |
| 回归评分 | 15% | OLS回归模型的统计显著性 |
| 稳健性评分 | 15% | 样本内外一致性和泛化能力 |

### 1.2 综合等级划分

| 等级 | 分数区间 | 含义 |
|------|----------|------|
| **A** | ≥ 0.75 | 优秀，因子质量极高 |
| **A-** | ≥ 0.60 | 良好偏上，因子有效 |
| **B** | ≥ 0.50 | 良好，因子基本可用 |
| **B-** | ≥ 0.45 | 一般偏上，需谨慎使用 |
| **C** | ≥ 0.30 | 一般，有效性存疑 |
| **D** | < 0.30 | 较差，不建议使用 |

## 二、各维度评分标准详解

### 2.1 IC评分（IC Score）

#### 2.1.1 核心指标

IC评分基于 **ICIR（信息比率）** 计算，公式为：

```
IC评分 = min(1.0, |ICIR|)
```

| ICIR区间 | 评分 | 评价 |
|----------|------|------|
| ≥ 1.0 | 1.00 | 满分，预测能力极强 |
| 0.8 ~ 1.0 | 0.80 ~ 1.00 | 优秀 |
| 0.5 ~ 0.8 | 0.50 ~ 0.80 | 良好，可接受 |
| 0.3 ~ 0.5 | 0.30 ~ 0.50 | 一般，偏弱 |
| < 0.3 | < 0.30 | 较差，预测能力弱 |

#### 2.1.2 IC检验相关指标

**基础指标**：
- **IC Mean**：滚动IC序列的均值，反映因子预测方向
- **IC Std**：滚动IC序列的标准差，反映稳定性
- **ICIR** = IC Mean / IC Std，信息比率，核心评分依据
- **IC > 0 占比**：IC序列中正值的占比

**统计显著性**：
- **T统计量**：检验IC均值是否显著偏离0
- **P值**：双侧检验p值
- **显著性**：p < 0.05 为显著

#### 2.1.3 IC参考标准

| 指标 | 优秀 | 良好 | 一般 | 较差 |
|------|------|------|------|------|
| ICIR | > 1.0 | > 0.5 | > 0.3 | ≤ 0.3 |
| IC Mean | > 0.05 | > 0.03 | > 0.01 | ≤ 0.01 |
| IC > 0 占比 | > 60% | > 55% | > 50% | ≤ 50% |

---

### 2.2 信号评分（Signal Score）

#### 2.2.1 评分逻辑

信号评分基于**胜率**和**统计显著性**综合计算：

```python
# 基础胜率得分（线性映射）
wr_score = min(1.0, max(0.0, (overall_win_rate - 0.50) / 0.20))

# 显著性加分
sig_bonus = 0.40 if is_significant else 0.20

# 最终得分
signal_score = min(1.0, wr_score + sig_bonus)
```

#### 2.2.2 胜率评分对照表

| 综合胜率 | 胜率得分 | 显著性加分 | 最高可能得分 |
|----------|----------|------------|--------------|
| ≥ 70% | 1.00 | +0.40 → 满分 | 1.00 |
| 65% | 0.75 | +0.40 | 1.00 |
| 60% | 0.50 | +0.40 | 0.90 |
| 55% | 0.25 | +0.40 | 0.65 |
| 50% | 0.00 | +0.40 | 0.40 |
| < 50% | 0.00 | +0.20 | 0.20 |

#### 2.2.3 信号生成方法

系统支持7种信号生成方法，取其中**最高分**作为最终信号评分：

| 方法 | 说明 | 信号生成规则 |
|------|------|--------------|
| **阈值法** | 基于因子极值 | 因子 > 上阈值(1.0) → 多头；因子 < 下阈值(-1.0) → 空头 |
| **均线法** | 基于因子趋势 | 因子 > N日均线 → 多头；因子 < N日均线 → 空头 |
| **极值法** | 基于历史分位数 | 因子 ≤ 低分位数(20%) → 多头；因子 ≥ 高分位数(80%) → 空头 |
| **零值法** | 基于零轴穿越 | 因子 > 均值(0) → 多头；因子 < 均值(0) → 空头 |
| **差分零值法** | 基于因子变化 | 因子差分 > 0 → 多头；因子差分 < 0 → 空头 |
| **MA250差分零值法** | 基于长期均线差分 | 250日均线差分 > 0 → 多头；反之空头 |
| **长短均线差值法** | 基于双均线差 | 短期均线 - 长期均线 > 0 → 多头；反之空头 |

#### 2.2.4 信号检验核心指标

**多头指标**：
- **n_long**：多头信号次数
- **long_win_rate**：多头胜率（收益率 > 0 的占比）
- **long_avg_return**：多头期间平均收益率
- **long_pl_ratio**：多头盈亏比（平均盈利/平均亏损绝对值）
- **long_coverage**：多头信号覆盖率

**空头指标**：
- **n_short**：空头信号次数
- **short_win_rate**：空头胜率（收益率 < 0 的占比）
- **short_avg_return**：空头期间平均收益率
- **short_pl_ratio**：空头盈亏比
- **short_coverage**：空头信号覆盖率

**综合指标**：
- **overall_win_rate**：综合胜率 = (正确多头 + 正确空头) / 总信号数
- **long_short_return_spread**：多空收益差
- **t_statistic**：多空收益率Welch t检验统计量
- **p_value**：双侧p值
- **is_significant**：p < 0.05 为显著

### 2.3 回归评分（Regression Score）

#### 2.3.1 评分逻辑

回归评分基于 **β系数显著性** 和 **R²拟合优度** 计算：

```python
# β显著性基础分
base = 0.50 if is_significant else 0.10

# R²加分（R²越高加分越多，最高加0.5）
r2_bonus = min(0.50, r_squared * 20)

# 最终得分
regression_score = min(1.0, base + r2_bonus)
```

#### 2.3.2 R²加分对照表

| R²值 | R²加分 | 说明 |
|------|--------|------|
| ≥ 0.025 | 0.50 | 满分加成 |
| 0.02 | 0.40 | 解释力强 |
| 0.015 | 0.30 | 解释力中等 |
| 0.01 | 0.20 | 解释力一般 |
| 0.005 | 0.10 | 解释力较弱 |
| < 0.005 | < 0.10 | 解释力很弱 |

#### 2.3.3 回归评分场景

| 场景 | β显著性 | R² | 基础分 | R²加分 | 总分 | 评价 |
|------|---------|-----|--------|--------|------|------|
| 理想 | 显著 | 0.03 | 0.50 | 0.50 | 1.00 | 满分，因子有效 |
| 良好 | 显著 | 0.02 | 0.50 | 0.40 | 0.90 | 因子有效，解释力中等 |
| 一般 | 显著 | 0.01 | 0.50 | 0.20 | 0.70 | 因子有效但解释力弱 |
| 边缘 | 不显著 | 0.05 | 0.10 | 0.50 | 0.60 | 可能有非线性关系 |
| 无效 | 不显著 | 0.005 | 0.10 | 0.10 | 0.20 | 线性关系不存在 |

#### 2.3.4 回归模型指标

**模型**：R(t+n) = α + β · F(t) + ε(t)

| 指标 | 说明 | 判断标准 |
|------|------|----------|
| **β** | 因子系数 | β > 0且显著：正向预测；β < 0且显著：反向预测 |
| **β_tstat** | β的t统计量 | 绝对值 > 2 通常认为显著 |
| **β_pvalue** | β的p值 | < 0.05 为显著 |
| **R²** | 拟合优度 | 解释收益率变动的比例 |
| **F_statistic** | F统计量 | 整体模型显著性检验 |

### 2.4 稳健性评分（Robustness Score）

#### 2.4.1 评分逻辑

稳健性评分评估因子在**样本外**的泛化能力：

```python
# 默认值（未运行检验时）
robustness_score = 0.50

# 运行检验后
same_sign = 1.0 if sign(样本外IC) == sign(样本内IC) else 0.0
deg_score = max(0.0, 1.0 - |IC衰减比率|)

robustness_score = 0.50 * same_sign + 0.50 * deg_score
```

#### 2.4.2 评分维度

| 维度 | 权重 | 说明 | 得分条件 |
|------|------|------|----------|
| **方向一致性** | 50% | 样本内外IC符号相同 | 同向得1.0，反向得0.0 |
| **IC衰减程度** | 50% | 样本外IC相对于样本内的衰减比例 | 无衰减得1.0，衰减100%得0.0 |

#### 2.4.3 稳健性判断标准

**判定为稳健（is_robust = True）的条件**：
1. 样本外IC与样本内IC符号相同（方向一致）
2. IC衰减幅度 < 50%（样本外IC ≥ 样本内IC的50%）

#### 2.4.4 场景示例

| 场景 | 样本内IC | 样本外IC | 同向得分 | 衰减得分 | 总分 | 稳健性 |
|------|----------|----------|----------|----------|------|--------|
| 理想 | 0.10 | 0.08 | 1.0 | 0.8 | 0.90 | ✓ 稳健 |
| 良好 | 0.10 | 0.05 | 1.0 | 0.5 | 0.75 | ✓ 稳健 |
| 临界 | 0.10 | 0.05 | 1.0 | 0.5 | 0.75 | ✓ 稳健 |
| 衰减大 | 0.10 | 0.03 | 1.0 | 0.7 | 0.85 | ✓ 稳健 |
| 方向反转 | 0.10 | -0.05 | 0.0 | 0.5 | 0.25 | ✗ 不稳健 |
| 严重衰减 | 0.10 | 0.01 | 1.0 | 0.9 | 0.95 | ✓ 稳健 |

#### 2.4.5 其他稳健性检验

**参数敏感性检验**：
- 在参数网格上遍历，考察因子表现对参数取值的依赖程度
- 稳健因子在参数小幅变动时表现应基本稳定

**市场状态检验**：
- 分别在牛市、熊市、震荡市中评估因子有效性
- 有助于理解因子的适用范围和失效条件
- 市场状态判定：基于滚动收益率（120日）
  - 牛市：滚动收益 > 15%
  - 熊市：滚动收益 < -15%
  - 震荡市：其余情形

## 三、综合评分计算流程

### 3.1 计算步骤

```
Step 1: 计算各维度得分
    ├── IC评分 = min(1.0, |ICIR|)
    ├── 信号评分 = min(1.0, 胜率得分 + 显著性加分)
    ├── 回归评分 = min(1.0, β基础分 + R²加分)
    └── 稳健性评分 = 0.5 × 同向得分 + 0.5 × 衰减得分

Step 2: 加权合成
    综合评分 = 0.35 × IC评分 + 0.35 × 信号评分 
             + 0.15 × 回归评分 + 0.15 × 稳健性评分

Step 3: 等级划分
    └── 根据分数区间确定 A/A-/B/B-/C/D 等级
```

### 3.2 评分示例

| 维度 | 原始值 | 得分 | 权重 | 加权贡献 |
|------|--------|------|------|----------|
| IC评分 | ICIR = 0.8 | 0.80 | 35% | 0.280 |
| 信号评分 | 胜率62%，显著 | 0.85 | 35% | 0.298 |
| 回归评分 | β显著，R²=0.02 | 0.90 | 15% | 0.135 |
| 稳健性评分 | 同向，衰减20% | 0.90 | 15% | 0.135 |
| **综合评分** | - | - | - | **0.848** |
| **等级** | - | - | - | **A** |

## 四、评分使用建议

### 4.1 各维度解读

| 维度高分 | 维度低分 | 建议 |
|----------|----------|------|
| IC评分高 | - | 因子具有预测能力，可关注 |
| 信号评分高 | - | 信号交易效果好，可直接用于交易 |
| 回归评分高 | - | 线性关系显著，模型可靠 |
| 稳健性评分高 | - | 泛化能力强，过拟合风险低 |
| - | IC评分低 | 因子预测能力弱，需优化或放弃 |
| - | 信号评分低 | 信号效果差，检查信号生成逻辑 |
| - | 稳健性低 | 存在过拟合，需简化模型或增加样本 |

### 4.2 综合决策矩阵

| 综合等级 | 建议操作 |
|----------|----------|
| **A (≥0.75)** | 强烈推荐使用，因子质量优秀 |
| **A- (≥0.60)** | 推荐使用，因子表现良好 |
| **B (≥0.50)** | 可以使用，需持续监控 |
| **B- (≥0.45)** | 谨慎使用，建议结合其他因子 |
| **C (≥0.30)** | 不建议单独使用，需大幅改进 |
| **D (<0.30)** | 不建议使用，因子无效 |

### 4.3 注意事项

1. **评分仅供参考**：评分是量化指标，实际使用需结合业务逻辑
2. **多周期验证**：建议检验多个预测周期（1日、5日、10日、20日）的表现
3. **样本量要求**：确保数据量充足，避免统计偏差
4. **市场适应性**：定期重新评估，市场环境变化可能导致因子失效
5. **组合使用**：单个因子评分不高时，可考虑多因子组合策略
