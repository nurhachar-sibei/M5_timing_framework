# 示例：A股择时因子评价

本示例将 `timing_framework` 包应用于四个常见择时因子，
数据为合成A股市场行情（特征接近沪深300指数）。

---

## 示例内容一览

| 章节 | 说明 |
|------|------|
| **数据生成** | 具有机制切换特征的合成A股价格序列 |
| **因子计算** | 基于价格/收益率序列计算四个择时因子 |
| **完整评估** | 每个因子通过信号检验 + IC + 回归 + 稳健性进行全面评估 |
| **横向对比** | ICIR、胜率、综合评分的多因子并列比较 |
| **市场状态分解** | 按牛市/熊市/震荡市分别评估因子质量 |
| **敏感性分析** | 在一组参数网格上评估MA动量因子 |
| **正交化** | 对称正交化前后的因子相关性对比 |

## 四个因子详解

### 1. MA动量因子

**公式**：`(MA_20 − MA_60) / MA_60`

衡量20日均价相对60日均价的偏离程度。

| 信号 | 条件 | 解读 |
|------|------|------|
| 做多（+1） | MA_20 > MA_60（因子 > 阈值） | 上升趋势——顺势持有 |
| 做空（−1） | MA_20 < MA_60（因子 < 阈值） | 下降趋势——空仓或做空 |

**最佳适用场景**：趋势性（动量型）市场。
**弱点**：在均值回归或震荡市场中容易反复被打止损。

---

### 2. RSI反转因子

**公式**：`(50 − RSI_14) / 50`

将标准14日RSI转化为反转择时信号。

| RSI值 | 因子值 | 信号 |
|-------|-------|------|
| < 30（超卖） | > 0.40 | 看多——逢低买入 |
| ≈ 50（中性） | ≈ 0    | 无明显信号 |
| > 70（超买） | < −0.40 | 看空——获利了结 |

**最佳适用场景**：均值回归型市场。
**弱点**：在动量行情中与趋势对抗，RSI可能长期维持极端值。

---

### 3. 波动率因子

**公式**：`−滚动标准差(收益率, 20)`

捕捉"低波动率异象"：高波动率时期通常与市场压力同步或预示下行风险。

| 波动率水平 | 因子值 | 信号 |
|-----------|-------|------|
| 低波动率 | 高（较小负值） | 市场平静 → 看多 |
| 高波动率 | 低（较大负值） | 市场动荡 → 看空 |

**最佳适用场景**：风险偏好切换时期。
**备注**：这是一个变化缓慢的防御性因子，在5–20日预测周期效果最佳。

---

### 4. 估值因子

**公式**：`−(PE − PE合理) / PE合理`，其中 `PE合理 = 15倍`

基于指数市盈率的均值回归因子。

| PE条件 | 因子值 | 信号 |
|--------|-------|------|
| PE < 15倍（便宜） | > 0 | 市场低估 → 看多 |
| PE ≈ 15倍（合理） | ≈ 0 | 无明显信号 |
| PE > 15倍（贵）   | < 0 | 市场高估 → 看空 |

**最佳适用场景**：长周期预测（10–20日以上），价值驱动型市场。
**备注**：本示例中的PE为模拟数据。实盘应使用Wind、Bloomberg或交易所的真实指数市盈率。

## 8子图评估看板解读

每个因子生成一张包含8个子图的评估图：

```
┌──────────────────────────┬──────────────────┐
│  因子值时序图             │  滚动IC时序图     │
│  （含±1σ阈值线）         │                  │
├─────────────┬────────────┼──────────────────┤
│  各信号平均  │  各方法    │  多预测周期       │
│  收益率      │  胜率对比  │  IC与ICIR        │
├─────────────┼────────────┼──────────────────┤
│  收益率分布  │  因子值vs  │  综合评分         │
│  （按信号）  │  未来收益  │  柱状图          │
└─────────────┴────────────┴──────────────────┘
```

**重点关注项**：

- **IC时序图**：应以正值为主，且较为平稳（避免剧烈波动）
- **各信号平均收益率**：做多柱应为正，做空柱应为负
- **胜率**：三种信号方法均应超过50%
- **综合评分柱**：好因子各维度均应为绿色（≥ 0.60）

## 合成数据结果解读

合成数据具有以下特征：
- 微小正向AR(1)系数 → MA动量因子应有正IC
- GARCH型波动率聚集 → 波动率因子应有效
- 机制切换收益率 → MA因子在趋势市场中表现更好

由于数据为随机生成（`seed=42`固定），各因子的表现会在统计上较为边缘，
但方向性上应基本正确。在真实A股数据中，这些规律通常更为显著。

## 接入真实数据

将示例中的合成数据生成器替换为真实行情数据即可：

```python
import pandas as pd

# 示例：使用AkShare（开源中国市场数据接口）
# pip install akshare
import akshare as ak

# 获取沪深300指数日线数据
df = ak.stock_zh_index_daily(symbol="sh000300")
df.index = pd.to_datetime(df["date"])
prices = df["close"].sort_index()
returns = prices.pct_change().dropna()

# 使用示例中相同的因子函数和评估器
factor = calc_ma_momentum(prices, 20, 60)

evaluator = TimingFactorEvaluator("MA动量", forward_period=1)
evaluator.evaluate(factor, returns, prices=prices)
evaluator.report()
```

## 从示例中汲取的关键经验

1. **预处理至关重要**：原始因子通常含有极端异常值，会严重扭曲所有下游指标。
   务必先进行MAD去极值处理。

2. **滚动窗口优于全样本**：在预处理中使用 `rolling_window=252`，
   可避免将未来信息引入回测。

3. **多方法交叉验证**：IC表现良好但信号胜率低的因子可能存在虚假相关。
   三种方法的结论应相互印证。

4. **稳健性是底线**：样本内IC在样本外衰减超过50%，说明因子很可能对历史噪声过拟合。

5. **关注市场状态**：在牛市中表现优异、在熊市中失效的因子依然有价值，
   与市场状态识别器配合使用即可。
